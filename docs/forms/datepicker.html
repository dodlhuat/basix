Date Picker

Calendar-based date picker that attaches to any text input. Supports single date, date ranges, an optional time picker, custom locales, and configurable week start day.

Usage

Options

OptionTypeDefaultDescription
modesingle | range'single'Single date or date range selection.
startDay0–60First day of the week. 0 = Sunday, 1 = Monday.
timePickerbooleanfalseShow hour and minute pickers below the calendar.
format(date: Date) => string—Custom display format function.
locales{ days: string[], months: string[] }EnglishLocalised day and month names.
selectedDateDatenullPre-selected date on init.
rangeStartDatenullPre-selected range start.
rangeEndDatenullPre-selected range end.
