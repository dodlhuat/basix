Color Picker

Canvas-based HSB color picker with a hue slider and hex/RGB inputs. Renders into any container and is fully mobile-friendly — drag on touch works without triggering page scroll.

{ ColorPicker } from '/basix/js/color-picker.js'

Usage

Options

OptionTypeDefaultDescription
valuestring'#ff0000'Initial hex color.
onChange(hex, rgb) => void—Called on every color change. Receives the hex string and an { r, g, b } object.

API

MethodDescription
getValue()Returns the current color as a lowercase hex string, e.g. '#3d63dd'.
setValue(hex)Sets the picker to a given hex color. Silently ignored if the string is not a valid 6-digit hex.
destroy()Removes all event listeners, disconnects the ResizeObserver, and clears the container DOM.
