Range Slider

Styled <input type="range"> with a filled-track accent that updates as the thumb moves. Wrap in .range-slider and initialise with RangeSlider to activate the fill.

Basic

Value: 50

With step

Value (step 25): 25

Usage

API

MethodDescription
update()Recalculates the fill percentage. Called automatically on input.
destroy()Removes the event listener and clears the CSS fill property.
RangeSlider.initAll(selector?)Initialises all matching inputs. Default: '.range-slider input[type="range"]'.
