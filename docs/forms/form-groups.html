Form Groups

Layout helpers for form controls: .form-group stacks a label, input, and hint text. .input-group attaches prefix or suffix addons. Validation states and size modifiers are purely CSS — no JS needed.

Form group

Wraps a label, input, and optional hint text into a consistent vertical layout.

Validation states

Add .error or .success to a .form-group to colour the border, icon, and hint text.

Input group

Attaches prefix or suffix addons to an input. Addons can contain text, icons, or buttons.
