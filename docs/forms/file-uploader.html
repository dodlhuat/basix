File Uploader

Drag-and-drop file upload zone with progress indication, file type and size validation, multi-file support, and upload cancellation. Fires custom events on completion and validation failure.

{ FileUploader } from '/basix/js/file-uploader.js'

Demo

Drag files onto the zone or click to browse. Accepted: PNG, JPG, GIF, SVG up to 5 MB.

Usage

Options

OptionTypeDefaultDescription
uploadUrlstring'https://httpbin.org/post'Endpoint that receives the multipart file upload.
maxFileSizenumber—Maximum file size in bytes. Files over this limit are rejected.
allowedTypesstring[]—MIME types allowed. Files with other types are rejected.

Events

EventDetailDescription
upload-completedPromiseSettledResult[]Fired on the container after all uploads finish.
file-validation-error{ file, reason: 'size' | 'type' }Fired when a file is rejected during selection.
