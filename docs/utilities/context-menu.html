Context Menu

Right-click context menu with nested submenus, separators, danger items, and full keyboard navigation (↑ ↓ Enter Escape). Attach to any element — the menu opens at cursor position.

Demo

Right-click here to open the context menu

Usage

Item schema

FieldTypeDescription
labelstringDisplay text of the menu item.
actionfunctionCalled when the item is clicked or activated via keyboard.
childrenItem[]Nested submenu items — renders a flyout panel on hover/focus.
type'separator'Renders a horizontal divider line instead of an interactive item.
dangerbooleanApplies a destructive red colour to the item label.
disabledbooleanRenders the item as non-interactive.

API

MethodDescription
destroy()Removes all event listeners from the target elements.

Keyboard navigation

KeyAction
↑ / ↓Move focus between items.
EnterActivate focused item.
EscapeClose the menu.
