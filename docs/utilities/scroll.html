Scroll

Smooth scroll utility for scrolling to any element by selector or DOM reference. Supports custom containers, scroll offset, and throttled scroll event listeners.

Usage

API

MethodDescription
Scroll.to(target, options?)Scrolls to the target element. target is a CSS selector or Element.
Scroll.toTop(options?)Scrolls the window back to the top of the page.
Scroll.onScroll(callback)Registers a throttled scroll listener. Returns an unsubscribe function.

Options

OptionTypeDefaultDescription
behavior'smooth' | 'instant' | 'auto''smooth'Scroll animation type — passed to scrollIntoView.
block'start' | 'center' | 'end' | 'nearest''start'Vertical alignment of the target after scrolling.
containerstring | ElementwindowScrollable container — defaults to the window.
