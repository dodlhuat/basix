Usage

Modifiers

Class on <table>Description
.tableRequired base class. Adds consistent borders, header styling, and hover row highlight.
.table-stripedAlternating row background for easier scanning of wide data sets.
.table-compactReduced cell padding for dense, information-heavy tables.
.table-borderlessRemoves all borders. Rows are separated by hover state and background only.

JS Data Table

The Table class renders a fully interactive data table from a JavaScript data array. It provides column sorting, a live search filter, and page-based pagination — all wired up automatically.

Interactive demo

The demo below has 15 rows with pageSize: 5. Try clicking a column header to sort, typing in the search box to filter, or navigating between pages.

JS usage

HTML usage

For programmatic initialisation, the only HTML you need is an empty container element.

Initialising from existing markup

If no data option is provided, the Table class will read column headers and row data directly from an existing <table> element inside the container, then replace it with the interactive version.

Options

OptionTypeDefaultDescription
dataTableRow[][]Array of row objects. Each object's keys must match the key values in columns.
columnsTableColumn[][]Column definitions controlling headers, sort keys, and sortability.
pageSizenumber10Number of rows shown per page. The user can also change this via the page-size selector.

Column definition

Each entry in the columns array is a TableColumn object:

FieldTypeDefaultDescription
keystring—Property name used to read values from each row object.
labelstring—Text shown in the column header.
sortablebooleantrueWhen true, clicking the header toggles ascending / descending sort on this column.

API

MethodDescription
setData(data)Replaces the current row data and re-renders the table from page 1.
setColumns(columns)Replaces the column definitions and fully rebuilds the table including controls.
getData()Returns the current data after applying any active search filter and sort order.
destroy()Removes all event listeners and clears the container element.
