Virtual Dropdown

Virtualised searchable dropdown for large datasets. Only the visible rows are rendered to the DOM — 10,000+ items scroll smoothly. Supports single and multi-select modes.

Usage

Options

OptionTypeDefaultDescription
containerstring | Element—CSS selector or DOM element to mount into.
options{ value, label }[][]Full option list — can be thousands of items.
placeholderstring'Select…'Text shown when nothing is selected.
searchablebooleanfalseShow a search input at the top of the list.
multiSelectbooleanfalseAllow selecting multiple items.
renderLimitnumber20Number of items rendered per virtual scroll window.
itemHeightnumber40Row height in px — must match CSS to calculate scroll positions correctly.
onSelectfunction—Called with (value, label) on selection change.
