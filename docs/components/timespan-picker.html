Timespan Picker

Time range picker for selecting a start and end time. Renders two time inputs with a visual duration bar that reflects the selected interval. Fires onChange with the start, end, and total minutes.

Demo

Selected: 09:00 → 17:00 (8h 0m)

Usage

Options

OptionTypeDefaultDescription
defaultStartstring—Pre-filled start time in HH:MM format.
defaultEndstring—Pre-filled end time in HH:MM format.
onChangefunction—Called whenever either time changes. Receives { start, end, durationMinutes }.
