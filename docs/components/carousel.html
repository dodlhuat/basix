Carousel

Touch-enabled slide carousel with prev/next controls, dot indicators, configurable loop and autoplay. Works with any content — images, cards, or custom HTML.

Usage

Options

OptionTypeDefaultDescription
loopbooleanfalseWraps from last slide back to first (and vice versa).
autoplaybooleanfalseAutomatically advances slides on an interval.
intervalnumber3000Milliseconds between auto-advances when autoplay is on.

API

MethodDescription
next()Advances to the next slide.
prev()Goes back to the previous slide.
goTo(index)Jumps to a specific slide by 0-based index.
pause()Pauses autoplay.
resume()Resumes autoplay.
destroy()Removes all event listeners and resets state.
