Group Picker

Hierarchical multi-select component with collapsible groups, search filtering, and a chip-based selection summary. Pass a tree of groups and subgroups — the picker handles the rest.

Usage

            


            
Data format

            

                
FieldTypeDescription

                
                    
idstringUnique identifier for the group or subgroup.

                    
labelstringDisplay name shown in the list.

                    
subgroupsarrayOptional array of child items (same shape without nesting).

                
            


            
Options

            

                
OptionTypeDefaultDescription

                
                    
onSelectionChangefunction—Called with { parents: Set, subs: Map } whenever selection changes.

                    
searchPlaceholderstring'Gruppen durchsuchen…'Placeholder text in the search input.

                    
selectAllLabelstring'Alle'Label for the "select all subgroups" action.

                    
deselectLabelstring'Abwählen'Label for the "deselect" action.

                    
emptyLabelstring'Keine Ergebnisse'Message shown when search yields no results.

                    
selectionPlaceholderstring'Noch keine Auswahl'Placeholder text in the chip area when nothing is selected.
