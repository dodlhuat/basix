Usage

Toolbar commands

Any button with a data-cmd attribute executes document.execCommand(cmd, false, value). The optional data-value attribute sets the command value.

data-cmdDescription
boldToggle bold on selection.
italicToggle italic on selection.
underlineToggle underline on selection.
strikeThroughToggle strikethrough on selection.
insertUnorderedListWrap in a bullet list.
insertOrderedListWrap in a numbered list.
justifyLeft / Center / RightText alignment.
foreColorApply text color — use data-value="#hex".
formatBlockWrap in block element — use data-value="h2" etc.
