Usage

Element attributes

All internal elements are located by scoped attribute queries within the root container — no globally unique IDs required.

data-editorElementRequired
editablecontenteditable div — writing areaalways
code<textarea> — HTML source viewfull mode
preview<div> — live previewfull mode
side-panelSide panel containerfull mode
code-panelCode tab panel (matches data-tab="code-panel")full mode
preview-panelPreview tab panelfull mode
wordcountWord count displayoptional
image-file<input type="file"> for image insertionoptional
data-editor-actionDescription
undoTriggers undo
redoTriggers redo
linkOpens URL prompt for link insertion
imageTriggers image file picker
saveDownloads content as HTML file (Ctrl+S)
clearClears all content
cleanStrips formatting from selection
toggle-codeShows / hides the HTML side panel

Toolbar commands

Any button with a data-cmd attribute executes document.execCommand(cmd, false, value). The optional data-value attribute sets the command value.

data-cmdDescription
boldToggle bold on selection.
italicToggle italic on selection.
underlineToggle underline on selection.
strikeThroughToggle strikethrough on selection.
insertUnorderedListWrap in a bullet list.
insertOrderedListWrap in a numbered list.
justifyLeft / Center / RightText alignment.
foreColorApply text color — use data-value="#hex".
formatBlockWrap in block element — use data-value="h2" etc.

Options

OptionTypeDefaultDescription
simplebooleanfalseSimple mode requires only [data-editor="editable"]. Full mode additionally requires code, preview, and side-panel elements.
rootstring | HTMLElementdocument.bodyRoot container for all scoped queries. Required when multiple editors exist on the same page.
