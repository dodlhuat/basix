Alerts

Contextual feedback messages. Five semantic variants — default, info, success, warning, error. Pure CSS with optional dismissal via a close button.

Variants

Default: A plain informational message.
Info: Some helpful information for you.
Success: Your changes were saved successfully.
Warning: Please review your input before continuing.
Error: Something went wrong. Please try again.

Dismissible

Info: Click × to dismiss this alert.
