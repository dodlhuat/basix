Scrollbar

Custom styled scrollbar using ::webkit-scrollbar pseudo-elements. Thin by default, expands on hover. Apply .custom-scrollbar to any scrollable container — no JavaScript required.

Usage

How it works

Pseudo-element / selectorPurpose
::-webkit-scrollbarSets scrollbar track width/height to a thin value (4 px).
::-webkit-scrollbar-trackTransparent track background.
::-webkit-scrollbar-thumbRounded thumb with a muted border color.
:hover::-webkit-scrollbarExpands track to 8 px on container hover for easier interaction.
scrollbar-width: thinFirefox equivalent — no pseudo-element support needed.

Note: ::-webkit-scrollbar is not a standard CSS property and works in Chromium-based browsers and Safari. Firefox uses scrollbar-width and scrollbar-color instead, which are included automatically via the .custom-scrollbar class.
