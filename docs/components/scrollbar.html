Scrollbar

Custom draggable scrollbar with a thin-at-rest, expands-on-hover thumb. Requires a small HTML structure and one new Scrollbar() call — the component handles thumb sizing, drag, track-click, and resize automatically.

{ Scrollbar } from '/basix/js/scrollbar.js'

Vertical scroll

Scroll this container to see the custom scrollbar. Hover near the right edge to reveal the thumb, or drag it directly.

Usage

API

MethodDescription
new Scrollbar(selector)Initialise on a .scroll-container element.
Scrollbar.initAll(selector)Initialise on all matching elements, returns array of instances.
destroy()Remove all event listeners and disconnect the ResizeObserver.
