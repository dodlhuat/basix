Gallery

Infinite-scroll masonry image gallery. Columns are computed from minColumnWidth, images are loaded via a custom fetchFunction, and new batches are fetched automatically as the user scrolls toward the bottom.

{ MasonryGallery } from '/basix/js/gallery.js'

Demo

Usage

Options

OptionTypeDefaultDescription
fetchFunction() => Promise<ImageData[]>—Required. Called on init and on each scroll trigger. Return an array of { src, title, desc } objects.
minColumnWidthnumber250Minimum column width in px — determines the number of columns.
scrollThresholdnumber100Distance from container bottom (px) that triggers the next fetch.
loaderSelectorstring'.loader'CSS selector for an optional loading indicator element.
