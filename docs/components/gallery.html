Gallery

CSS grid image gallery with optional lightbox on click. Images load lazily with the native loading="lazy" attribute. The Gallery class intercepts clicks to open a full-screen lightbox with keyboard navigation.

Demo

Usage

Lightbox controls

InteractionAction
Click imageOpens lightbox on that image.
← → Arrow keysNavigate to previous / next image.
EscapeClose the lightbox.
Click overlayClose the lightbox.
