Calendar

A full-featured calendar component with month, week, and agenda views. Supports events, all-day spans, keyboard navigation, and dark mode.

{ Calendar } from '/basix/js/calendar.js'

Month view

Monthly grid with up to three event pills per day and a "+n more" overflow indicator.

Week view

7-column time grid with absolute-positioned timed events and an all-day row at the top.

Agenda view

Compact list of days that have events, grouped by day with full time ranges.
