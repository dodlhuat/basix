Chat Bubbles

Message bubble layout for chat interfaces with incoming and outgoing styles, avatar support, and timestamps. Pure CSS — no JavaScript required.

Usage

Classes

ClassElementDescription
.chatcontainerFlex column wrapping all messages.
.chat-messagerowSingle message row — flex with gap.
.incoming.chat-messageAligns bubble to the left (other person).
.outgoing.chat-messageAligns bubble to the right (current user).
.chat-avatardiv / imgCircular avatar shown on incoming messages.
.chat-bubbledivThe styled bubble background with padding and border-radius.
.chat-time<time>Small muted timestamp rendered below the message text.
