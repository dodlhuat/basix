Stepper

Multi-step process indicator with horizontal and vertical layouts. Steps can be completed, active, or in an error state. The Stepper class manages transitions programmatically.

Usage

Step states

ClassDescription
.completedStep has been successfully completed.
.activeCurrent active step.
.errorStep encountered a validation error.
.stepper-verticalSwitches to a vertical layout with description support.

API

MethodDescription
next()Marks the current step as completed and advances to the next.
prev()Returns to the previous step.
goTo(index)Jumps directly to a step by 0-based index.
setError(index)Applies the error state to the specified step.
clearError(index)Removes the error state from the specified step.
destroy()Removes event listeners and resets state.
