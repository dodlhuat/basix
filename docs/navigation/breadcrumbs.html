Breadcrumbs

Hierarchical navigation trail built from a semantic <ol> inside .breadcrumb. Pure CSS — separators are rendered via ::before pseudo-elements. No JavaScript required.

Default

  1. Home
  2. Documentation
  3. Breadcrumbs

Deeper nesting

  1. Dashboard
  2. Users
  3. John Doe
  4. Edit Profile

Usage

Accessibility

Use aria-label="Breadcrumb" on the <nav> element to identify the landmark. Mark the current page with aria-current="page" on its <li>. Use a <span> (not an <a>) for the current item since it links to the current page.
