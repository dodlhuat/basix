Sidebar Nav

An always-visible sidebar on desktop that collapses into a slide-in drawer on mobile. Zero configuration — just add the CSS classes and call new SidebarNav().

{ SidebarNav } from '/basix/js/sidebar-nav.js'

How it works

Wrap your page in .sidebar-layout. Place the sidebar as .sidebar-nav, a backdrop overlay as .sidebar-backdrop, and the rest of the page inside .sidebar-main. On mobile a hamburger button with class .sidebar-toggle opens the drawer.

HTML structure

JavaScript

Pass the layout container and an optional options object. The component wires up the toggle button, backdrop click-to-close, and auto-close on resize.

Options

API reference

MethodDescription
open()Slide the sidebar in and show the backdrop.
close()Slide the sidebar out and hide the backdrop.
toggle()Toggle between open and closed states.
isOpen()Returns true if the sidebar is currently open.
destroy()Remove all event listeners added by this instance.

Options reference

OptionTypeDefaultDescription
toggleSelectorstring'.sidebar-toggle'CSS selector for the hamburger toggle button.
breakpointnumber768Viewport width (px) above which the sidebar is permanently visible and the drawer auto-closes.

CSS classes

ClassElementDescription
.sidebar-layoutWrapperFlex container that holds the sidebar and main content side by side.
.sidebar-nav<aside>The sidebar panel. Sticky on desktop, fixed drawer on mobile.
.sidebar-backdrop<div>Full-screen semi-transparent overlay shown behind the open drawer.
.sidebar-main<div>The scrollable main content area next to the sidebar.
.sidebar-toggle<button>Hamburger button — hidden on desktop, visible on mobile.
.is-open.sidebar-navAdded by JS when the drawer is open.
.is-visible.sidebar-backdropAdded by JS when the backdrop is shown.
