Flyout Menu

A slide-in navigation panel that overlays the page from the left or right edge. Supports nested submenus, optional header and footer, keyboard Escape to close, and overlay-click dismissal.

Demo

Usage

Options

OptionTypeDefaultDescription
triggerSelectorstring'.menu-trigger'Elements that open the menu on click.
menuSelectorstring'#flyoutMenu'The flyout panel element.
overlaySelectorstring'#flyoutOverlay'The backdrop overlay element.
closeSelectorstring'.close-menu'Elements inside the menu that close it on click.
directionright | left'right'Side the panel slides in from.
titlestring'Menu'Header title text.
enableHeaderbooleantrueShow the menu header with title.
footerTextstring'© 2025 Brand Inc.'Footer text content.
enableFooterbooleantrueShow the menu footer.

API

MethodDescription
open()Opens the flyout panel.
close()Closes the flyout panel.
destroy()Removes all event listeners.
