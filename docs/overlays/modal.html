Modal

Dialog overlays with header, content, and footer sections. Supports semantic types, a close button, backdrop and Escape key dismissal. Built entirely in JavaScript — no HTML markup required.

Variants

Usage

Options

OptionTypeDefaultDescription
contentstring—HTML string for the modal body.
headerstring—HTML string for the modal header.
footerstring—HTML string for the modal footer, typically action buttons.
typedefault | success | error | warning | info'default'Applies a semantic colour to the modal header.
closeablebooleantrueShows × button and enables backdrop/Escape dismissal.
