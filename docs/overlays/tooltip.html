Tooltip

Shows contextual information on hover or focus. Supports four explicit placements plus auto-detection, HTML content, and both declarative (data-tooltip) and programmatic APIs.

Placement

HTML content

Declarative

Add data-tooltip and optionally data-tooltip-position to any element, then call Tooltip.initializeAll().

Usage

Options

OptionTypeDefaultDescription
positiontop | bottom | left | right | auto'auto'Preferred placement. auto picks the side with the most space.
offsetnumber8Gap in px between trigger and tooltip.
delaynumber0Show delay in ms.
classNamestring''Extra CSS class added to the tooltip element.

API

MethodDescription
show()Shows the tooltip immediately (respects delay).
hide()Hides the tooltip.
destroy()Removes the tooltip and all event listeners.
Tooltip.initializeAll()Initialises all [data-tooltip] elements on the page.
