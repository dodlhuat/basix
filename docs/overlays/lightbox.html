Lightbox

Fullscreen image viewer with gallery navigation, keyboard and swipe support, click-to-zoom, and declarative HTML binding. Built entirely in JavaScript — no markup required.

{ Lightbox } from '/basix/js/lightbox.js'

Single image & gallery

Declarative binding

Call Lightbox.bind() to attach click handlers to elements with data-lightbox. Elements sharing the same attribute value are grouped into a gallery.

Options

OptionTypeDefaultDescription
srcstring—Image URL for single-image mode.
altstring—Alt text for the image.
captionstring—Optional caption rendered below the image.
imagesLightboxImage[]—Array of images for gallery mode. Each item: { src, alt?, caption? }. Overrides src/alt/caption.
startIndexnumber0Index of the image to show first in gallery mode.
closeablebooleantrueShows × button and enables backdrop click / Escape dismissal.
onOpenfunction—Callback fired when the lightbox opens.
onClosefunction—Callback fired after the close animation completes.

Methods

MethodDescription
show()Opens the lightbox.
hide()Closes the lightbox.
next()Navigates to the next image in gallery mode.
prev()Navigates to the previous image in gallery mode.
isVisible()Returns true if the lightbox is currently open.
destroy()Closes and removes the lightbox.
Lightbox.bind(selector?)Attaches click handlers to all matching elements. Default selector: [data-lightbox]. Groups elements by their data-lightbox value into galleries.

Keyboard & interaction

InputAction
EscapeClose the lightbox (when closeable).
→ / ←Navigate to next / previous image in gallery.
Tab / Shift+TabCycle focus within the lightbox (focus trap).
Click imageToggle zoom (fit ↔ actual size).
Swipe left / rightNavigate gallery on touch devices.
Click backdropClose the lightbox (when closeable).
